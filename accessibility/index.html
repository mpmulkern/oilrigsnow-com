Accessibility Statement

Published: May 22, 2026  •  Last Reviewed: May 22, 2026

Oil Rigs Now, LLC is committed to ensuring that new.oilrigsnow.com is accessible to all users, including people with disabilities. We are working toward conformance with the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA and compliance with the Americans with Disabilities Act (ADA) and Section 508 of the Rehabilitation Act.

1. Conformance Status

This website is partially conformant with WCAG 2.1 Level AA. "Partially conformant" means that some parts of the content do not fully conform to the accessibility standard, and we are actively working to address known gaps.

Our target standard: Web Content Accessibility Guidelines (WCAG) 2.1, Level AA

2. Accessibility Features Implemented

We have built the following accessibility features into this site:

Navigation & Structure

  • Implemented  Skip-to-content link on every page, allowing keyboard users to bypass the navigation
  • Implemented  Logical heading hierarchy (H1 → H2 → H3) on all pages
  • Implemented  Landmark regions (header, main, nav, footer) for screen reader navigation
  • Implemented  Descriptive page titles and breadcrumb navigation
  • Implemented  ARIA labels on icon-only buttons and navigation regions

Visual Design

  • Implemented  Text sized in relative units (rem/em) that scale with browser font size settings
  • Implemented  Status badges use both color and text (not color alone) to convey information
  • Implemented  Focus indicators visible on all interactive elements (links, buttons, form controls)
  • In Progress  Color contrast verification for all text and UI elements against WCAG 4.5:1 ratio requirement

Images & Media

  • Implemented  Alt text on all informational images
  • Implemented  Decorative images marked with empty alt attributes

Forms & Interactivity

  • Implemented  All form inputs have associated labels
  • Implemented  Full keyboard navigation for all interactive elements
  • Implemented  Search and filter controls on the browse page are keyboard accessible

Language & Compatibility

  • Implemented  lang attribute set on all pages
  • Implemented  RTL (right-to-left) layout support when Arabic language is selected
  • Implemented  Compatible with screen readers including NVDA, JAWS, and VoiceOver
  • Implemented  Compatible with keyboard-only navigation and browser zoom up to 400%

3. Known Limitations

Despite our best efforts, some content may not yet fully meet WCAG 2.1 AA standards. Known limitations include:

  • Equipment listing PDFs: Some third-party PDF documents linked from equipment pages may not be fully accessible. We are working to provide accessible HTML alternatives.
  • Third-party embedded content: Any third-party widgets (such as the AI chat assistant) are subject to the accessibility practices of the third-party provider. We evaluate accessibility as part of our vendor selection process.
  • Legacy listing pages: Older individual equipment pages may have lower accessibility conformance than newly built pages. We are reviewing and updating these systematically.

We are actively working to resolve known issues. If you encounter a specific accessibility barrier not listed here, please tell us (see Section 5 below).

4. Technical Approach

This website is built as a static site using semantic HTML5. We follow these technical standards:

  • Valid, semantic HTML5 markup
  • CSS that does not convey meaning (meaning is in the HTML)
  • JavaScript that enhances but does not replace keyboard or screen reader access
  • Responsive design that adapts to screen sizes from 320px to 4K
  • No time limits on any content or interactions
  • No content that flashes more than 3 times per second
