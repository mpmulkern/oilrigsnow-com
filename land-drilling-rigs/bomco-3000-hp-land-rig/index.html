Key Advantages

01

Full API Standards Compliance

Every major structural component carries the API monogram and is manufactured to current editions: API Spec 4F-2020 (5th Ed.) for mast and substructure; API Spec 7K-2015 (6th Ed.) for drawworks, rotary table, and BOP handling; API Spec 8C-2012 (5th Ed.) for crown block, traveling block, hook, swivel, and elevator links; and API Spec 9A-2020 (27th Ed.) for wire rope. Hazardous area classification follows API RP505-2018.

02

9,000 m Rated Drilling Depth

The ZJ90DB package is engineered for deep well work to 9,000 m with a 6,750 kN (1,500,000 lb) rated hook load at 14-line 7×8 reeving, a 48.7 m K-type mast, and triple 2,400 HP quintuplex mud pumps delivering 51.7 MPa (7,500 psi) working pressure. The 37-1/2" rotary table accommodates the full collar and casing program.

03

AC-VFD Drive Train with Siemens PLC

All primary loads — drawworks, rotary table, top drive, and mud pumps — are driven one-to-one via AC–VFD–AC with ABB drive cabinets. The control system uses Siemens S7-1500 PLC with dual-CPU cold-standby redundancy. The driller's cabin features a touch-screen interface, CCTV integration, and real-time parameter monitoring displayed at three locations (driller, toolpusher, client rep).

04

15,000 PSI BOP & Complete Well Control Stack

The well control package spans three pressure ratings: a 13-5/8" 15K stack with shear-blind rams rated for H2S service, a 21-1/4" 10K stack, and a 26-3/4" 3K diverter system. A 12-station API-16D accumulator unit with dual electric/air power and a 20,000 psi triplex BOP test pump complete the assembly. Total well control section value: $4.6M.

05

Dual 50/60 Hz Power Architecture

Five CAT 3512/3512C generators at 1,310 kW each supply 600V/50 Hz for main equipment. A Volvo TAD1651GE auxiliary unit serves camp and ancillary loads. Three additional 400 kW Volvo sets operate at 60 Hz, with 2+1 redundancy, to supply PEMEX/third-party equipment via a dedicated MCC house — enabling seamless integration with client-supplied cementing units, solids control, and trailers.

06

Turnkey Package with Drill String, Camp & Support

The quotation covers a complete turnkey scope: full drill string (2-7/8" through 5-7/8" S135 and G105 grades, 22,000+ m total), eight-unit camp accommodation (PEMEX and Chinese crews), MSA gas detection and SCBA safety systems, Bomay integrated drilling instrumentation, and BOMCO field service (8 personnel) for installation, commissioning, and 14-day operator training. Packaged FOB Tianjin Port in 6–8 months.
